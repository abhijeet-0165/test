Question 13  [Intermediate]  Arrays - Search Element
Question:
Read an array from user input. Search for a specific element entered by the user and print its index, or print 'Not Found' if it does not exist.
Solution:
#!/bin/bash
read -a arr
read target
found=0
for i in "${!arr[@]}"; do
    if [[ "${arr[$i]}" == "$target" ]]; then
        echo "Found at index $i"
        found=1
        break
    fi
done
(( found == 0 )) && echo "Not Found"

Question 16  [Intermediate]  Shell Functions - add and factorial
Question:
Write a shell script defining two functions: add() that accepts two arguments and prints their sum, and factorial() that accepts one argument and prints its factorial. Call them from stub code.
Solution:
#!/bin/bash
add() {
    local n1=$1
    local n2=$2
    echo $(( n1 + n2 ))
}
factorial() {
    local num=$1
    local fact=1
    for (( i=1; i<=num; i++ )); do
        fact=$(( fact * i ))
    done
    echo $fact
}

Question 17  [Intermediate]  Menu-Driven Script
Question:
Write a menu-driven shell script using case with options: 1) Add two numbers, 2) Check even/odd, 3) Print current SHELL, 4 or default) Print 'Invalid choice'.
Solution:
#!/bin/bash
read ch
case $ch in
    1)
        read n1 n2
        echo "Sum=$(( n1 + n2 ))"
        ;;
    2)
        read n
        if (( n % 2 == 0 )); then
            echo "Even number"
        else
            echo "Odd number"
        fi
        ;;
    3)
        echo "$SHELL"
        ;;
    *)
        echo "Invalid choice"
        ;;
esac

Question 18  [Intermediate]  Nested Loops - Number Pattern
Question:
Write a shell script using nested loops to print the following number pattern. Accept the number of rows n from the user. For n=3: row 1 prints 1, row 2 prints 12, row 3 prints 123.
Solution:
#!/bin/bash
read n
for (( i=1; i<=n; i++ )); do
    for (( j=1; j<=i; j++ )); do
        echo -n "$j"
    done
    echo
done

Question 21  [Advanced]  Arrays - Menu-Driven Program
Question:
Write a menu-driven shell script that reads all elements of an array at once, then offers: 1) Display all elements, 2) Display the 3rd element, 3) Display the total count, 4) Add a new element to the end, default) Display all elements.
Solution:
#!/bin/bash
read -a arr
read ch
case $ch in
    1) echo "${arr[@]}" ;;
    2) echo "${arr[2]}" ;;
    3) echo "${#arr[@]}" ;;
    4)
        read x
        arr+=($x)
        echo "${arr[@]}"
        ;;
    *) echo "${arr[@]}" ;;
esac

Question 24  [Advanced]  systemd Timer - Create a Timer Unit
Question:
Create a systemd service unit and a corresponding timer unit to run a script /opt/report.sh every 5 minutes. Show the unit file contents and the commands to enable and start the timer.
Solution:
# /etc/systemd/system/report.service
[Unit]
Description=Run report script

[Service]
ExecStart=/opt/report.sh


# /etc/systemd/system/report.timer
[Unit]
Description=Run report every 5 minutes

[Timer]
OnBootSec=5min
OnUnitActiveSec=5min
Unit=report.service

[Install]
WantedBy=timers.target


# Enable and start
sudo systemctl daemon-reload
sudo systemctl enable report.timer
sudo systemctl start report.timer
systemctl list-timers
